Gallery of a single dir
galerie --gallery ./singledir --sourcedir ./medias

Gallery of a single dir extended with sub directories
galerie --gallery ./singledir --sourcedir ./medias --recursive true --github_pages true

Gallery organized by directories
galerie --gallery ./bydir --sourcedir ./medias --bydir true --github_pages true

Gallery organized by date
galerie --gallery ./bydate --sourcedir ./medias --bydate true --github_pages true

Gallery based on diary
galerie --gallery ./diary --diary true --github_pages true

Gallery based on diary extended with the content of a media directory
galerie --gallery ./diary_extended --sourcedir ./medias --github_pages true
